# circle-and-cross

マルとバツを使った、簡単で奥深い心理戦。

## ルール

４枚パネルが配られ、バツが1枚、マルが3枚。 
ランダムに親を決める。


まず全員、4枚のパネルから1枚を選んで伏せる。 

全員がパネルを1枚置いた時点で、親プレイヤーは以下の2つの選択肢から選ぶ 
→「パネルをさらに1枚重ねる」
→「勝負に出る」
 
○パネルをさらに1枚重ねる
手元のパネルから更に1枚選んで重ねる。 
手番は時計回りで隣のプレイヤーに動き、隣のプレイヤーもまた、「パネルをさらに1枚重ねる」か「勝負に出る」かの選択ができる。 
（パネルが手元になく、「パネルを重ねる」ことができない場合、必ず「勝負に出る」必要がある） 
 
○勝負に出る
プレイヤーは任意の自然数を宣言する。 
数字を宣言されたら、各プレイヤーは時計回りに、以下の選択を迫られる 
→「パス」
→「それより大きい数字を宣言する」 
 
・パス
パスをしたらその時点でこの勝負から降りることを宣言する。 

・それより大きい数字を宣言する
新たな宣言者となる。

宣言者以外、全員がパスをした時点でめくり始める。
めくるルールは以下のとおり。
→「まず自分のパネルをすべてめくる」 
→「一番上のパネルから順番にめくる」 

 
最終的に無事、バツをめくることなくマルを宣言枚数めくることができれば1勝。 
2勝した時点で勝利。 
もし、途中でバツをめくった場合、敗者となり、4枚のパネルを1枚え選び失う。 
パネルが0枚になったら、ゲームから脱落
